# Python Automation

This is a simple Python Automation project that automatically opens multiple websites in the browser based on the selected category.

## Features

- Open multiple websites automatically
- Command line argument support
- Separate categories for work and personal use
- Simple and beginner-friendly Python project

## Technologies Used

- Python
- webbrowser module
- sys module
